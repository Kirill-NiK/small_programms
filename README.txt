Можно добавлять как файлы, так и промежуточные пути (например, /thirdparty/trololo), они не будут включены в посчет.
Пути надо добавлять через '/'. Путь типа "D:\QReal\qreal\" исключен не будет.
Путь папки для подсчета должен заканчиваться на '/'.
Подсчет только по методам, которые имеют реализацию в .cpp. (например, не идет подсчет в интерфейсах)
